python3 manager2.py

how to use
connect with the barcode reader. Barcode reader works same as a keyboard. It inputs code and presss enter
database is in the excel file

software can update the new product, price and name to excel file
it also can update old name and price

ctrl-L for update info
ctrl-R for reset info, start with new customer
up down arrow keys for access to quantity
